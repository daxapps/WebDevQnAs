# WebDevQnAs

## Summary
The app will help web developers study interview questions with flashcards.

## Technology Used
* HTML/CSS/jQuery
* Node
* Express
* MongoDB w/Mongoose
* Passport (authentication)
* Mocha/Chai (testing)
* Travis CI (integration)
* Deployed with Heroku
